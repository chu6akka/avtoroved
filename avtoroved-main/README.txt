==========================================================
  АВТОРОВЕДЧЕСКИЙ АНАЛИЗАТОР v5
  Инструмент судебно-автороведческой экспертизы текста
==========================================================

УСТАНОВКА
---------
1. Установите Python 3.10+ (https://python.org)
2. Откройте терминал/командную строку в папке программы
3. Установите зависимости:
      pip install -r requirements.txt
4. При первом запуске Stanza автоматически скачает модель
   русского языка (~100 МБ) — нужен интернет.

ЗАПУСК
------
      python autoroved_gui.py

СОСТАВ ФАЙЛОВ
-------------
  autoroved_gui.py          — главный файл программы
  lexical_stratification.py — модуль стратификации лексики
  lexicon_stratified.json   — словарная база (12 251 лемма)
  analyzer/errors.py        — модуль анализа ошибок и навыков
  analyzer/__init__.py      — служебный файл
  requirements.txt          — список зависимостей

ВКЛАДКИ ПРОГРАММЫ
-----------------
  1. Морфология              — POS-теги, леммы, морф. признаки
  2. Статистика              — TTR, длины, POS-биграммы
  3. Ошибки и навыки         — степени развития навыков (Вул)
  4. Интернет-коммуникация   — признаки сетевого идиолекта
  5. Стратификация лексики   — распределение по стилистич. пластам
  6. Сравнение текстов       — идентификационная задача
  7. Сводный отчёт           — экспорт в DOCX
