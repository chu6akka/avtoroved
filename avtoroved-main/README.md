# Автороведческий анализатор v5

Инструмент судебно-автороведческой экспертизы текста на русском языке.
Превосходит Lingster 3.0 по функционалу, реализован на PyQt6.

## Возможности

| # | Функция | Описание |
|---|---------|----------|
| 1 | **Морфология** | POS-теги, леммы, морф. признаки (Stanford Stanza) |
| 2 | **Статистика** | TTR, hapax, POS-биграммы (Litvinova et al., 2015–2016) |
| 3 | **Ошибки и навыки** | 4 категории ошибок, степени по С.М. Вул (2007) |
| 4 | **Интернет-коммуникация** | 12 маркеров сетевого идиолекта |
| 5 | **Стратификация лексики** | 13 пластов, 12 251 лемма (ЭКЦ МВД России, 2021) |
| 6 | **Сравнение текстов** | 6 метрик сходства: Jaccard + POS + TTR + биграммы |
| 7 | **ИИ-детектор (GigaCheck)** | Локальная модель SberDevices, подсветка интервалов |
| 8 | **Тематические словари** | 10 областей: право, медицина, IT, экономика и др. |
| 9 | **Грамматические запросы** | Поиск паттернов `NOUN ADJ`, `VERB * NOUN`, wildcards |
| 10 | **Пакетная обработка** | Несколько файлов за раз, DOCX-отчёт |
| 11 | **Экспорт** | Сводный отчёт в DOCX, сравнительный отчёт |

## Установка

```bash
pip install -r requirements.txt
```

При первом запуске Stanza автоматически скачает модель русского языка (~100 МБ).

## Запуск

```bash
python main.py
```

```

```

> **GigaCheck** работает локально без интернета после первой загрузки модели (~1–4 ГБ).
> Результаты GigaCheck носят вспомогательный характер и не заменяют лингвистический анализ.

## Методическая основа

- Назарова Л.С., Маняни В.А., Громова А.Н. «Методика автороведческой экспертизы» (ЭКЦ МВД России, 2021)
- Вул С.М. «Судебно-автороведческая идентификационная экспертиза» (2007)
- Litvinova et al. (2015–2016) — POS-биграммы для профилирования автора
- «Правила русской орфографии и пунктуации» (АН СССР, 1956)
- Химик В.В. «Большой словарь русской разговорной экспрессивной речи» (2004)
- GigaCheck — SberDevices, [arXiv:2410.23728](https://arxiv.org/abs/2410.23728)

## Требования к системе

- Python 3.10+
- ОС: Windows / macOS / Linux
- RAM: 2 ГБ+ (без GigaCheck), 8 ГБ+ (с GigaCheck)
- GPU (опционально): ускоряет работу GigaCheck
